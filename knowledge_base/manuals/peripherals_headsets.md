## 1. Catalog Overview
The TechNova headset lineup covers entry-level wired, mid-range wireless, premium wireless ANC, and pro-grade wired ANC. All headsets are certified for Microsoft Teams and Zoom; mid-range and above add Google Meet and Webex certification.

The detailed manual for the flagship **TechNova Premier Wireless Headset HS-720** is published in MAN-ACC-CAT-2025 (accessories.md). This document covers the rest of the lineup.

## 2. TechNova Essential Stereo Headset HS-310

| Spec | Value |
|---|---|
| Connectivity | USB-A wired (3 m cable) |
| Battery | N/A (USB-powered) |
| Microphone | Boom (non-retractable), unidirectional |
| Drivers | 28 mm neodymium |
| Frequency response | 20 Hz - 20 kHz |
| Certifications | Microsoft Teams, Zoom |
| Weight | 165 g |

### Use Cases
- Call centers, classroom labs, kiosk and shared-desk environments.
- In-line mute and volume controls; no driver installation required.

### Setup
1. Plug into any USB-A port on a Windows, macOS, Linux, or Chrome OS device.
2. The OS detects the headset as "TechNova HS-310" within 5 seconds.
3. Set as default audio device under `Settings > Sound > Input/Output`.

| Spec | Value |
|---|---|
| Connectivity | Bluetooth 5.3, 2.4 GHz USB-A dongle, USB-C wired |
| Battery | Up to 35 hours (Bluetooth), 28 hours (dongle) |
| Charging | USB-C, full charge in 100 minutes |
| Microphone | Retractable boom, beamforming dual-mic |
| Drivers | 40 mm neodymium |
| Active Noise Cancellation | Passive only (no ANC) |
| Certifications | Microsoft Teams, Zoom, Google Meet |
| Weight | 220 g |

### First-time pairing (HS-520)
1. Slide the power switch to **On**. Hold the **Bluetooth** button 3 seconds until the LED flashes blue/red.
2. On your computer or phone, select `TechNova HS-520` from the Bluetooth list.
3. To use the dongle, slide the source switch from **BT** to **Dongle**; the dongle auto-pairs.

## 7. Driver and Firmware Updates
- Open **TechNova Peripheral Manager (TPM)** on Windows or macOS.
- Connect the headset via USB (wired or charging cable) for firmware updates. Firmware updates over Bluetooth are **not** supported.
- TPM check schedule: weekly automatic, manual via `Help > Check for Updates`.

## 8. Warranty
All TechNova headsets carry a **1-year limited warranty** (3-year optional via TechNova Accessory Care). See WARRANTY-POL-001. Ear cushions and ear pads are classed as consumables and are not covered after 90 days from purchase.
